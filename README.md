# AI 導入專案儀表板

一個純前端、可編輯的專案管理儀表板，用來追蹤公司七階段 AI 導入計畫的進度。

## 專案背景

本專案規劃 9 個月內（2026/04 - 2026/12）分七個階段導入 AI，減少重複性工作：

1. **階段 1：客服**（2026/04 - 2026/05）- AI 全自動接手，不補人力
2. **階段 2：通路（Line 禮物 + 團媽）**（2026/05 - 2026/06）- 訂單、對帳、訊息自動化
3. **階段 3：採購**（2026/07）- 比價、進貨單、庫存預測
4. **階段 4：會計**（2026/08）- 發票辨識、分類記帳、報表
5. **階段 5：HR + 全專案回顧**（2026/09 - 2026/10）- 員工問答、教育訓練、ROI 回顧
6. **階段 6：生產部**（2026/11）- 生產排程、品管記錄、產能預測
7. **階段 7：直營部**（2026/12）- 門市營運、排班、CRM、銷售分析

## 功能

- 整體進度總覽與七階段 KPI 卡片
- 甘特圖呈現時程
- 每個階段可展開/收合，內含目標、成效指標、任務清單、風險
- 所有文字可直接點擊編輯
- 任務狀態三段切換（待辦 → 進行中 → 完成）
- 新增／刪除任務
- 匯出／匯入 JSON 保存進度
- 列印模式（一鍵輸出 PDF）
- 響應式設計，手機可用

## 使用方式

直接打開 `index.html` 即可使用，不需要任何安裝步驟。

- 點擊任何文字可直接編輯
- 點擊任務左側圓圈切換狀態
- 編輯完成後按「匯出 JSON」保存
- 下次再按「匯入 JSON」接續進度

## 技術堆疊

- 純 HTML + CSS + 原生 JavaScript
- 零依賴，單一檔案
- 約 700 行，可直接嵌入任何靜態站台
